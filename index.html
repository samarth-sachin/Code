üß† The Backend Brain: Java & AI-Powered Decision Making

Core Logic Flow

The backend's primary responsibility is to orchestrate the secure wiping process intelligently. On device detection, it triggers AI analysis using DeepLearning4J to classify the device type, assess data sensitivity, and determine optimal wiping strategy. Each device's wiping state is tracked using blockchain transactions for immutable audit trails.

The backend is structured into microservices for scalability: the Controller handles API requests, the AI Service performs device analysis and risk assessment, the Wiping Engine executes secure deletion algorithms, and the Blockchain Service manages smart contract verification.

The AI Decision Tree

The AI uses trained machine learning models to create an intelligent decision path for each device type and data classification. This provides context-aware wiping strategies rather than simple one-size-fits-all approaches.
